Adds a setting in General Settings that allows you to customize the greeting in your admin panel.

== Description ==

This filters the admin bar, allowing you to change the default "Howdy" greeting to be whatever you want it to be. It adds a section in Settings > General allowing you to set whatever greeting you would like, limited to 30 characters.

== Installation ==

Upload the Admin Greeting Customizer plugin to your blog, Activate it, then enter your desired greeting in Settings > General.

Easy Peasy.
